# ProjectAVG
This project is part of subject Problem Solving in Information Technology (PSIT)

## Objective
Project นี้มีวัตถุประสงค์จัดทำขึ้นเพื่อให้ผู้ที่สนใจได้ศึกษา วิเคราะห์ข้อมูลเกี่ยวกับรายได้เฉลี่ยต่อเดือน - ต่อครัวเรือน จำแนกตามภาค 
และจังหวัด พ.ศ. 2541 - 2558 ซึ่งรวบรวมรายได้ทั้งหมดของครัวเรือน ทั้งรายได้จากอาชีพหลัก รายได้จากอาชีพรอง อาชีพเสริม รายได้อื่น ๆ และรายได้ที่เกิดจากการทำ การปลูก การเลี้ยงและการหาไว้กินเอง (แล้วคิดคำนวณเป็นค่าเฉลี่ยของแต่ละปี)
